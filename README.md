1. copy files riemannian_leapfrog.py, visualization_dashboard.py, geometry_of_awareness.py, and visualization_ui.py into a directory of your choosing,
2. in the terminal navigate to the directory, then enter: streamlit run visualization_ui.py
3. A locally hosted version will start at local host 8501.
4. on the left hand sidebar, pick your variables and seed, then click Generate.
5. The generated images and an interpretation of what the results are will be provided
